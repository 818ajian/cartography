barradeau.com

cartography

a series of examples around creative cartography in 2D and 3D, using THREE.js and custom tools.

you'll need a local server to load some tile sets (elevation especially)

2D tile systems

  • cutting a big picture into XY tiles
  • using a .mbtiles file to store tiles for offline use

2D vectors

  • basic GPS tracking data: plotting a month of my personal data collected from my google Dashboard
  • clustering GPS data: using alpha shapes to cluster data visually
  • aggregating GPS data: plotting a day of 5 random San Francisco cabs GPS data (source)
  • filtering GPS data: removing noise and indexing colors on the length of each segment

3D elevation (warning this will not work online)

  • basic GPS tracking data: a month of my personal data collected from my google Dashboard
  • clustering GPS data: using alpha shapes to group data visually
